 Job Search App 

A full-stack **Job Search Application** developed using the **MERN stack** that allows users to search, apply, and manage job listings with secure authentication and role-based access.

---

 Features

- User authentication (Login & Signup)
- Role-based access (Admin & User)
- Browse and search job listings
- Apply for jobs
- Admin can add, update, and delete jobs
- REST API based architecture
- Responsive user interface

---

 Tech Stack

**Frontend**
- React
- Redux
- Chakra UI
- JavaScript

**Backend**
- Node.js
- Express.js
- MongoDB (MongoDB Atlas)
- JWT Authentication
